# Sprinkler Control System - Evolutionary Algorithm Optimization

## Project Objectives

Design a simulation of an **Automatic Sprinkler Control System** to demonstrate the efficiency of evolutionary algorithms in computational parameter optimization.

---

## Project Description

This project is a **Soft Computing** course assignment that aims to:
- Implement an automatic sprinkler control system
- Use evolutionary algorithms (Genetic Algorithm, Particle Swarm Optimization, or similar)
- Optimize system parameters based on historical data
- Compare system performance before and after optimization

## Setup & Installation

### Prerequisites
- Python 3.8+
- `uv` package manager

### Install UV Package Manager

If you don't have `uv` installed, follow the official installation guide:
- **Official Documentation:** [https://docs.astral.sh/uv/getting-started/installation/](https://docs.astral.sh/uv/getting-started/installation/)

**Quick Install:**
```bash
# Windows (using pip)
pip install uv

# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Install Dependencies
```bash
uv sync
```
